# Costco Sales and Profitability Analysis Dashboard

## 📊 Project Overview
This is an **Exploratory Data Analysis (EDA)** and interactive dashboard built in **Microsoft Excel** for the Costco Sales dataset. The dashboard provides clear insights into sales performance, profitability, regional trends, top performers, and product categories across the United States from 2014.

The analysis helps identify:
- High-performing salespersons and regions
- Peak sales periods and product categories
- Customer and shipping trends
- Overall revenue and profitability drivers

**Tools Used**: Microsoft Excel (PivotTables, Slicers, Charts, Conditional Formatting)

**Dataset**: Costco Sales data (commonly available on Kaggle or similar sources)

## 🎯 Key Insights from the Dashboard
- **Total Revenue**: $437,066
- **Top Salesperson**: Nancy
- **Best Performing Region**: New York (Regional Lead by Revenue)
- **Peak Month**: December 2014
- **Top Product Category by Revenue**: Beverages
- **Top Ship Cities**: New York, Miami, Memphis, etc.

The dashboard includes interactive slicers for easy filtering by year, product category, region, and more.
